# Function Generator

---

## Overview

A **Function Generator** (also called a signal generator or waveform generator) is an electronic test instrument that produces electrical waveforms of various shapes, frequencies, and amplitudes on demand. It is the "signal source" of the electronics laboratory — providing the controlled, well-characterized test signals needed to evaluate, characterize, and troubleshoot electronic circuits, sensors, and systems.

The name "function generator" comes from the mathematical functions that define the most common waveforms it produces: sine, square, and triangle (or ramp) waves. Modern function generators are more accurately called **Arbitrary Waveform Generators (AWGs)** when they can generate user-defined waveform shapes in addition to standard functions.

In an electronics laboratory, a function generator is almost always used in combination with an oscilloscope: the generator provides the stimulus to the circuit under test, and the oscilloscope displays the circuit's response. Together, they are the foundation of analog circuit characterization.

Function generators are used across virtually every area of electrical and electronics engineering:
- Amplifier frequency response testing (Bode plots)
- Filter characterization
- ADC and DAC testing
- Signal processing algorithm development
- Sensor calibration
- Audio circuit testing
- Communication system testing
- Motor control waveform generation

> **Assumed Specifications (General Lab Function Generator):**
> - Waveforms: Sine, Square, Triangle, Ramp (Sawtooth), Pulse, Noise, and Arbitrary (AWG)
> - Frequency Range: 0.1 Hz to 30 MHz (for sine/square); varies by model
> - Amplitude: 1 mVpp to 20 Vpp (into 50 Ω or open circuit)
> - DC Offset: ±10 V
> - Frequency Resolution: 1 µHz (on DDS models)
> - Output Impedance: 50 Ω
> - Waveform Symmetry (Duty Cycle): 1% to 99% (for square/pulse)
> - Modulation: AM, FM, PM, FSK, PWM, Sweep
> - Channel: 1 to 2 independent channels
> - Display: Color LCD with waveform preview and parameter display
> - Interfaces: USB, LAN (optional), GPIB (optional)

## Working Principle

### Direct Digital Synthesis (DDS)

Modern function generators use **Direct Digital Synthesis (DDS)** technology, which is fundamentally different from older analog oscillator-based designs. DDS is a digital technique for generating waveforms from stored numerical data, providing exceptional frequency accuracy, resolution, and agility.

**How DDS works:**

1. **Phase Accumulator:** A digital counter (phase accumulator) increments by a fixed value (the phase increment word, Δφ) every clock cycle. The increment value determines the frequency of the output waveform:

   > f_out = (Δφ × f_clock) / 2^N
   
   Where N is the bit width of the phase accumulator (e.g., 32 bits) and f_clock is the master clock frequency.

2. **Waveform Lookup Table (LUT):** The output of the phase accumulator (the current phase) is used as an address into a Read-Only Memory (ROM) lookup table that stores a digital representation of one complete waveform cycle (e.g., one period of a sine wave, sampled at many points).

3. **DAC (Digital-to-Analog Converter):** The digital value read from the lookup table is converted to an analog voltage by a high-speed DAC. This creates a "staircase" approximation of the desired waveform.

4. **Reconstruction Filter (Low-Pass Filter):** The staircase DAC output contains high-frequency components (images) from the digital sampling process. A low-pass reconstruction filter smooths the staircase into a clean analog waveform.

5. **Output Amplifier and Attenuator:** The reconstructed waveform is amplified or attenuated to the desired output amplitude, and a DC offset is added if required. The output is then routed to the 50 Ω output terminal.

### Frequency Accuracy and Agility

Because DDS frequency is determined by the digital phase increment word (a programmable register), frequency can be changed instantly and with extremely fine resolution (typically 1 µHz or better) without any analog retuning. This also makes DDS ideal for frequency sweep, modulation, and multi-tone signal generation.

### Square Wave Generation

Square waves are not typically generated from a sine lookup table. They are generated by a digital comparator on the phase accumulator output, creating a precise 50% (or variable) duty cycle square wave with very fast rise and fall times determined by the digital logic speed.

### Arbitrary Waveform Generation (AWG)

In AWG mode, the lookup table ROM is replaced with user-writable RAM. The user downloads a custom waveform sample sequence (up to thousands of points) into this RAM, and the DDS engine plays it back at the selected clock rate. This allows generation of completely custom waveforms — cardiac pulse shapes, sensor simulation signals, distorted waveforms, etc.

## Main Components

| Component | Description |
|---|---|
| **Master Clock / Reference Oscillator** | High-stability crystal oscillator (often 10 MHz, 100 MHz, or higher) that clocks the DDS engine. Accuracy of the clock determines frequency accuracy. |
| **Phase Accumulator** | High-resolution digital accumulator (24–48 bits) that counts phase linearly; determines output frequency. |
| **Waveform RAM/ROM** | Stores the digital samples of one waveform period; ROM for standard waveforms, RAM for arbitrary. |
| **DAC (High-Speed DAC)** | Converts digital waveform samples to analog voltage; key determinant of waveform fidelity. |
| **Reconstruction Filter** | Low-pass anti-imaging filter to smooth DAC staircase output. |
| **Output Amplifier** | Amplifies the reconstructed waveform to the desired amplitude. |
| **Attenuator** | Precision resistive attenuator to reduce amplitude for small output levels. |
| **DC Offset Circuit** | Adds a selectable DC offset to the output waveform. |
| **Output Impedance Network** | Sets the output impedance to 50 Ω for compatibility with standard RF test equipment. |
| **Trigger Input** | External trigger input (BNC) for synchronizing output to an external event. |
| **Sync / TTL Output** | Digital TTL-level square wave output at the same frequency as the main output; useful for triggering an oscilloscope. |
| **Modulation Circuits** | AM, FM, PM, FSK modulation inputs and control circuits for generating modulated signals. |
| **Microcontroller / FPGA** | Controls all functions, processes user input, drives the display, manages communications. |
| **LCD Display** | Shows current waveform shape, frequency, amplitude, offset, and other parameters. |
| **Front Panel Keypad and Knobs** | FREQUENCY knob, AMPLITUDE knob, numeric keypad, waveform selection buttons. |
| **USB/LAN Interface** | For remote control and arbitrary waveform download from a computer. |

## Inputs and Outputs

### Inputs
- **AC Mains Power:** For the generator itself.
- **External Reference Clock Input (10 MHz):** For synchronizing the DDS clock to an external reference (lab frequency standard) for maximum accuracy.
- **External Trigger/Gate Input:** BNC input; triggers waveform output, burst mode, or sweep.
- **Modulation Input (AM/FM):** External analog signal for amplitude or frequency modulation.
- **USB/LAN:** For arbitrary waveform data download and remote control.

### Outputs
- **Main Output (BNC, 50 Ω):** Primary waveform output; labeled as OUTPUT or CH1.
- **Sync / TTL Output (BNC):** Digital TTL square wave at same frequency as the main output; used to trigger the oscilloscope.
- **Second Channel Output (if dual channel):** Independent secondary waveform output.

## Operating Procedure

1. **Power on the function generator.** The display will show the current output settings (or default settings after reset).

2. **Select the waveform type** using the WAVEFORM button or touchscreen:
   - Sine, Square, Triangle, Ramp, Pulse, or Noise.

3. **Set the frequency** using the FREQUENCY knob or numeric keypad. Use the unit buttons (Hz, kHz, MHz) to set the appropriate unit. For example, to set 1 kHz:
   - Press FREQ, type 1, press kHz.

4. **Set the amplitude** using the AMPLITUDE knob or numeric keypad. Specify peak-to-peak (Vpp) or RMS (Vrms) as available:
   - Press AMPL, type 1, press Vpp.

5. **Set the DC offset** if required. Default is 0V (centered around zero). To shift the waveform:
   - Press OFFSET, enter the desired offset value in volts.

6. **For square waves, set the duty cycle** (the percentage of the period during which the output is at high level):
   - Select square wave; press DUTY CYCLE or equivalent; enter value (e.g., 25% for a 25% duty cycle).

7. **Enable the output** by pressing the OUTPUT ON button. On many generators, the output is disabled at power-on for safety. A button press enables it and the OUTPUT indicator LED lights up.

8. **Connect the output to your circuit** using a BNC-to-probe cable or BNC-to-banana cable, with the center conductor to the signal terminal and the outer shield to ground.

9. **Connect the SYNC output to the oscilloscope's external trigger input** (or use CH1/CH2 of the scope connected to the generator output) to obtain a stable triggered display on the oscilloscope.

10. **To set up a frequency sweep (for Bode plot characterization):**
    - Select SWEEP mode.
    - Set Start Frequency, Stop Frequency, and Sweep Time.
    - Connect the SYNC output to the oscilloscope trigger.

11. **To generate an arbitrary waveform:**
    - Use the manufacturer's PC software to design or import a waveform.
    - Download it to the generator via USB.
    - Select ARB mode on the generator.

12. **Turn output OFF before making major parameter changes** to avoid sending incorrect signals to your circuit during adjustment.

## Safety Guidelines

- **Check the circuit's maximum allowable input voltage** before connecting the generator output. High amplitudes can destroy logic IC inputs, op-amps, or other components.
- **The output is 50 Ω source impedance.** If the load is not 50 Ω, the actual voltage at the load will differ from the display setting. Into an open circuit (high impedance like an oscilloscope), the actual peak-to-peak voltage will be approximately 2× the stated value (because the 50 Ω output and 10 MΩ scope input form a nearly unity-gain voltage divider — all voltage appears at the load). Into a true 50 Ω load, the voltage will match the display. Always verify with the oscilloscope.
- **Do not connect the generator output directly to a low-impedance source** (like a power supply output) — this can cause large currents to flow and damage the generator's output stage.
- **Use proper BNC cables and connectors.** Loose or damaged BNC connections cause signal reflections, especially at high frequencies.
- **At high frequencies (>10 MHz), use 50 Ω coaxial cables** to prevent signal reflections from impedance mismatches.
- **Avoid generating very high voltages (>10 Vpp) into unknown loads.** While the generator limits current via the 50 Ω output resistance, some situations can still cause component damage.
- **The generator output is not isolated from the chassis ground** (signal ground = earth ground via mains). Be aware of this when working with floating or isolated circuits.

## Typical Applications

- **Amplifier testing:** Apply a sine wave input; measure output amplitude vs. frequency to characterize gain and bandwidth (Bode magnitude plot).
- **Filter characterization:** Apply a swept sine wave; observe the output amplitude roll-off to determine cutoff frequency and filter order.
- **ADC testing:** Apply a sine wave of known frequency and amplitude to an ADC input; analyze the digitized output for quantization noise, SNR, ENOB (Effective Number of Bits).
- **Clock signal generation:** Provide a precise square wave clock for synchronous digital circuits during prototyping.
- **PWM generation:** Generate variable-duty-cycle square waves for motor speed control or LED brightness testing.
- **Resonance demonstrations:** Apply a swept frequency to an RLC circuit; observe the peak response at the resonant frequency.
- **Sensor simulation:** Generate waveforms that simulate the output of a sensor (e.g., a pseudo-ECG signal for medical device testing).
- **Modulation experiments:** Generate AM/FM modulated signals; demodulate using a circuit and verify the recovered baseband signal.
- **Communication system testing:** Generate standard data patterns for communication link testing.
- **Audio circuit testing:** Generate audio-frequency sine and square waves to test amplifiers, speakers, and filters.

## Common Student Projects

1. **Frequency Response (Bode Plot) Measurement:** Apply a sine wave to an RC low-pass filter; measure output amplitude and phase at each frequency (10 Hz to 10 MHz in decades); plot gain (dB) vs. frequency; compare to theoretical Bode plot.

2. **Square Wave Spectrum Analysis:** Generate a square wave; display it on the oscilloscope; use the FFT function to observe the harmonic content (odd harmonics at 1f, 3f, 5f, 7f...); verify that a square wave is the sum of sine wave harmonics.

3. **RC Circuit Time Constant Verification:** Apply a square wave (slow enough for the capacitor to fully charge/discharge); observe the exponential charging/discharging waveform on the oscilloscope; measure the time constant τ and compare to calculated R×C.

4. **AM Modulation Generation and Demodulation:** Use the generator's AM mode to create an amplitude-modulated signal; build an envelope detector (diode + RC filter); verify the demodulated signal recovers the original modulating frequency.

5. **Lissajous Phase Measurement:** Set CH1 of the oscilloscope to the generator output and CH2 to the output of a phase-shifting network (RC circuit); use XY mode on the oscilloscope to display Lissajous figures at various frequencies; calculate phase from the ellipse parameters.

6. **PWM Motor Speed Controller Lab:** Generate PWM square waves at fixed frequency with varying duty cycle; apply to a DC motor driver circuit; measure motor speed vs. duty cycle using a tachometer or encoder.

7. **RLC Resonance Experiment:** Connect a series RLC circuit to the generator; sweep frequency through the resonant frequency; observe the voltage across the resistor peak at resonance; measure bandwidth, calculate Q factor.

8. **Arbitrary Waveform Cardiac Signal Simulation:** Download a real ECG waveform (from a public database) and import into the generator's PC software; download to the generator; use the ARB output to simulate a heartbeat signal for testing a medical device input amplifier.

9. **DAC Characterization:** Apply a precision sine wave from the function generator to an ADC; digitize and reconstruct using a DAC; observe the reconstructed output on the oscilloscope; calculate SINAD, ENOB, and THD.

10. **FSK Signal Generation and Decoding:** Configure the generator in FSK (Frequency Shift Keying) mode with two frequencies representing logic 0 and logic 1; observe the FSK signal on the oscilloscope; manually decode the transmitted bit pattern from the frequency transitions.

## Common Mistakes

- **Not accounting for output impedance (50 Ω):** The most common misunderstanding. When connected to a high-impedance input (oscilloscope, 10 MΩ), the actual output voltage is nearly double what the generator's amplitude setting shows (because the 50 Ω output resistance has almost no voltage drop). Always verify the actual voltage with the oscilloscope. When connected to a 50 Ω load (terminator), the voltage will be exactly as set.

- **Forgetting to enable the output:** Many generators have the output disabled at power-on and require pressing OUTPUT ON explicitly. Students often spend several frustrating minutes "debugging" their circuit when the generator simply has its output turned off.

- **Setting DC offset without accounting for amplitude:** If amplitude is set to 10 Vpp and offset to +5V, the positive peak is +10V and the negative peak is 0V. Forgetting this can inadvertently forward-bias components that should be reverse-biased.

- **Using too high a frequency without regard for the circuit's bandwidth:** Applying a 10 MHz signal to a circuit with a 100 kHz bandwidth and expecting to see 10 MHz behavior at the output — the circuit will attenuate the signal heavily.

- **Measuring amplitude with incorrect oscilloscope probe settings:** Using a 1x probe when the scope expects 10x (or vice versa) gives amplitude readings that are 10x too high or too low. Always match probe attenuation to the scope's channel setting.

- **Impedance mismatch at high frequencies:** Using regular banana-plug cables or unshielded wires at frequencies above 1–10 MHz causes significant signal reflections and distortion. Always use 50 Ω coaxial cables with BNC connectors for RF-range signal work.

- **Incorrect duty cycle for square wave characterization:** When using a square wave to test rise time, the period must be much longer than the rise time to allow the signal to fully reach its high and low levels before the next transition.

- **Forgetting to set back to safe defaults after high-amplitude testing:** Leaving the generator at 20 Vpp can destroy the next circuit connected.

## Maintenance Basics

- **Inspect BNC connectors and output terminals** for bent pins, corrosion, or damaged threads. A damaged BNC connector on the generator output must be repaired to maintain signal integrity.
- **Keep the front panel clean** — dust can accumulate in the keypad and knob mechanisms; a soft brush or compressed air can clean it.
- **Check the calibration output** periodically — the 1 kHz CAL square wave (if present) should match its rated amplitude and frequency; significant deviation suggests calibration drift.
- **Annual calibration** by a qualified metrology service is recommended to ensure frequency accuracy (especially of the reference oscillator) and amplitude accuracy across all ranges.
- **Update firmware** from the manufacturer's website — firmware updates can fix bugs in waveform generation, add features, or improve frequency accuracy.
- **Store BNC-to-BNC cables coiled** (not kinked) to preserve the coaxial structure and maintain characteristic impedance.
- **Do not disconnect the output under a reactive load** — disconnecting the cable to an inductive load (like a coil) can cause voltage spikes at the output.

## Troubleshooting Guide

| Problem | Possible Cause | Suggested Action |
|---|---|---|
| No output signal visible on oscilloscope | Output not enabled; cable/connection issue; wrong oscilloscope channel | Press OUTPUT ON; check BNC cable; select correct scope channel |
| Output amplitude is half of set value | Load is 50 Ω; generator is set for open-circuit calibration | This is correct behavior; the 50 Ω load creates a voltage divider with the 50 Ω source impedance |
| Output amplitude is twice expected | Load is high-impedance (oscilloscope 1 MΩ input); generator was calibrated for 50 Ω | Verify actual amplitude with oscilloscope; use 50 Ω terminator if needed |
| Waveform looks distorted at high frequency | Exceeded generator bandwidth; cable impedance mismatch | Use within specified frequency range; use 50 Ω coaxial cable |
| Frequency is slightly off target | Calibration drift; external reference not locked | Recalibrate; lock to 10 MHz external reference |
| Square wave has poor rise/fall times | Cable capacitance loading; probe capacitance; bandwidth limitation | Use 50 Ω coaxial cable; use 10:1 probe; check bandwidth |
| Modulation output does not work | Modulation not enabled in settings; wrong modulation source selected | Enable modulation; select correct modulation source (internal or external) |
| Arbitrary waveform not loading from PC | Software version mismatch; USB driver issue; incorrect waveform file format | Update software; reinstall USB driver; use correct AWG file format |
| Generator output is silent (DC level only) | Amplitude set to 0; DC offset equals amplitude | Increase amplitude; check amplitude and offset settings |
| Display shows no communication with PC | Wrong COM port; SCPI not enabled; firewall blocking LAN | Select correct interface in software; enable SCPI/LAN in generator settings; check firewall |

## Frequently Asked Questions

**Q1: What is Direct Digital Synthesis (DDS) and why is it used in modern function generators?**
A: DDS is a digital technique for generating waveforms. A phase accumulator counts linearly, and its output addresses a waveform lookup table stored in RAM/ROM. The digital samples are converted to analog by a DAC and filtered. DDS provides extremely fine frequency resolution (µHz), fast frequency switching, excellent accuracy, and phase-continuous frequency changes — all impossible with traditional analog oscillator designs.

**Q2: What is the difference between a function generator and an arbitrary waveform generator (AWG)?**
A: A function generator produces standard mathematical waveforms (sine, square, triangle, ramp). An AWG can generate any user-defined waveform by allowing the user to specify the waveform point-by-point. Most modern instruments are both — they have standard waveforms plus an ARB mode with downloadable RAM for custom shapes.

**Q3: What does "50 Ω output impedance" mean and how does it affect my measurements?**
A: The generator's output has a 50 Ω source resistance in series with the waveform. When loaded with a 50 Ω termination (e.g., a 50 Ω coaxial load), the voltage at the load is exactly half the open-circuit voltage — this is the "matched" condition where the generator's display calibration is accurate. Into a high-impedance load (oscilloscope 10 MΩ), the load gets essentially all the open-circuit voltage, which is about 2× the displayed value (assuming the generator was calibrated for 50 Ω loading). Always verify with your oscilloscope.

**Q4: What is a "burst" mode on a function generator?**
A: Burst mode outputs a specified number of complete waveform cycles when triggered, then stops and waits for the next trigger. For example, burst mode set to 5 cycles outputs exactly 5 sine wave cycles per trigger pulse. Useful for simulating radar pulses, ultrasonic bursts, or discrete packet signals.

**Q5: What is frequency sweep mode?**
A: Sweep mode automatically and continuously changes the output frequency between a start frequency and a stop frequency over a specified time. Linear or logarithmic sweep patterns are typically available. This is the basis for performing Bode plot (frequency response) measurements when the oscilloscope amplitude is plotted vs. sweep time (which corresponds to frequency).

**Q6: What is the SYNC output and how should I use it?**
A: The SYNC output is a TTL-level square wave at the same frequency as the main output. It is used to provide a trigger signal to the oscilloscope without using the main output signal itself as a trigger. This is useful for burst signals, swept signals, or when you want a clean trigger reference separate from the measured signal.

**Q7: What is AM modulation mode on a function generator?**
A: In AM (Amplitude Modulation) mode, the output signal's amplitude is varied by a modulating signal (either internal or applied to an external input). The result is a carrier signal whose amplitude envelopes a lower-frequency modulating waveform — the same as a radio AM signal. The displayed output shows the carrier with a time-varying amplitude envelope.

**Q8: What is FM modulation mode?**
A: In FM (Frequency Modulation) mode, the instantaneous frequency of the carrier varies in proportion to the modulating signal's amplitude. The frequency swings above and below the carrier frequency by an amount equal to the set deviation. FM is used in radio broadcasting and many analog communication systems.

**Q9: What is FSK (Frequency Shift Keying)?**
A: FSK is a digital modulation scheme where two (or more) distinct frequencies represent digital logic states (0 and 1). The generator switches between the two frequencies according to a digital data pattern. It is used in modems, RFID, and many wireless communication systems.

**Q10: What is PWM (Pulse Width Modulation) mode on a function generator?**
A: In PWM mode, a pulse (square wave) output has its duty cycle modulated by a control signal. The pulse frequency is fixed but the on/off ratio varies. PWM is used in motor control, DC-DC converter control, LED dimming, and DAC-free analog control applications.

**Q11: How do I generate a signal without any DC offset?**
A: Set the DC Offset to 0V (the default). In this state, a 2 Vpp sine wave swings from –1V to +1V, centered on zero.

**Q12: How do I generate a unipolar signal (all positive voltages)?**
A: Set the DC offset equal to half the amplitude. For example, for a 2 Vpp sine wave ranging from 0V to +2V, set amplitude to 2 Vpp and offset to +1V.

**Q13: What is the Nyquist frequency limit for a DDS generator?**
A: A DDS system with a clock frequency f_clock can theoretically generate output frequencies up to f_clock / 2 (Nyquist limit). In practice, the output reconstruction filter and DDS linearity limit usable output to about 40% of the clock frequency. So a 100 MHz clock DDS system can generate clean signals up to approximately 40 MHz.

**Q14: What is waveform symmetry / duty cycle on a function generator?**
A: For a square or pulse wave, duty cycle is the percentage of the period that the signal is in the high (positive) state. 50% duty cycle = equal high and low times. For a triangle/ramp wave, symmetry controls the ratio of rise time to fall time — 50% symmetry = symmetric triangle; 0% or 100% symmetry = sawtooth wave with instantaneous rise or fall.

**Q15: What SCPI commands can I use to control the function generator from Python?**
A: Using the PyVISA library, common SCPI commands include:
- `FUNC SIN` — select sine waveform
- `FREQ 1000` — set frequency to 1000 Hz
- `VOLT 1.0` — set amplitude to 1 Vpp
- `VOLT:OFFS 0` — set DC offset to 0V
- `OUTP ON` — enable output
- `OUTP OFF` — disable output

Each manufacturer may have slightly different SCPI command syntax — always refer to the instrument's programming guide.

**Q16: What is phase offset and how do I use it in a dual-channel generator?**
A: Phase offset sets the phase difference between two signals. On a dual-channel generator, CH1 might output a reference sine wave (0° phase), while CH2 outputs the same frequency but phase-shifted by a user-specified amount (e.g., 90°). This is useful for generating I and Q signals for quadrature modulation tests, or for simulating phase differences in power systems.

**Q17: Can a function generator damage my circuit?**
A: Yes — if you apply higher voltages than the circuit's components are rated for, or if you drive a very low impedance load (which the 50 Ω output cannot handle without current limiting), damage can occur. Always check your circuit's absolute maximum input voltage and the expected load impedance before connecting the generator.

**Q18: What is a Bode plot and how do I create one using a function generator?**
A: A Bode plot shows a circuit's frequency response — gain (output/input ratio in dB) and phase shift — as a function of frequency. To create one: apply a known-amplitude sine wave from the function generator, measure the output amplitude with the oscilloscope at each frequency, calculate gain = 20×log10(Vout/Vin), and plot vs. frequency on a log scale. Repeat for each decade of frequency from well below to well above the expected cutoff frequency.

## Related Machines

- **Oscilloscope** — The primary companion instrument; used to observe and measure all waveforms generated by the function generator.
- **Variable Power Supply** — Powers the circuit under test while the function generator provides the signal input.
- **Digital Multimeter (DMM)** — Measures RMS voltage and frequency of the generator output at low frequencies; less useful than the oscilloscope for waveform observation.
- **Spectrum Analyzer** — Analyzes the frequency domain content of the generator output; more capable than the oscilloscope's FFT for RF signal quality assessment.
- **Network Analyzer (VNA)** — More sophisticated instrument for measuring S-parameters and impedance; automates Bode plot creation internally.
- **Frequency Counter** — Precisely measures the frequency of the generator output; used to verify generator calibration.
- **Signal Analyzer / VSA (Vector Signal Analyzer)** — Analyzes complex modulated signals (AM, FM, QAM, etc.) generated by the function generator in modulation mode.

## Learning Path

**Beginner:**
- Generate a sine wave and observe it on an oscilloscope; adjust frequency and amplitude.
- Practice using the TIME/DIV on the oscilloscope to show 2–3 complete cycles.
- Generate square and triangle waves; observe their differences from a sine wave.
- Measure frequency and amplitude from the oscilloscope display and compare to generator settings.
- Understand the 50 Ω impedance issue by measuring the output into a high-impedance scope vs. a 50 Ω terminator.

**Intermediate:**
- Set a DC offset on the waveform; verify with the oscilloscope.
- Use the sweep mode to perform a Bode plot of an RC filter.
- Generate a square wave and use FFT on the oscilloscope to observe harmonic content.
- Set up burst mode and single-shot oscilloscope triggering to capture one burst event.
- Use FM and AM modulation modes; observe and interpret the resulting waveforms.

**Advanced:**
- Create custom arbitrary waveforms using the generator's PC software; download and output them.
- Control the function generator remotely using Python and PyVISA (SCPI commands).
- Perform a complete frequency response characterization (Bode plot) of an active filter, comparing it to theory.
- Design a two-tone test signal (using dual-channel AWG); measure intermodulation distortion of an amplifier.
- Implement a hardware-in-the-loop simulation using the AWG to simulate sensor outputs for testing embedded control systems.
